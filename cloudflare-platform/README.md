# ZeazDev Cloudflare Platform

Production-grade, phased Cloudflare Zero Trust platform for `zeaz.dev`.

## Phases
- F1 Context + Variables
- F2 Terraform Foundation
- F3 Zero Trust + Identity
- F4 DNS + Tunnels + Networking
- F5 Workers + Edge + AI
- F6 Monitoring + DR + Security

## Quick start
```bash
make validate
make plan ENV=dev
```
