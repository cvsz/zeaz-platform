# First 100 Users Growth Loop (Safe Mode)

## Objective

Activate the first 100 real Thai Shopee Affiliate creators with a manual-first, privacy-preserving workflow that prioritizes compliance and trust.

## Target user profile

- Thai Shopee Affiliate creators managing content manually
- Small sellers and creators who need faster caption and post workflows
- Users who want Thai-first UX and clear step-by-step onboarding
- Users who want optional short promo videos via HyperFrames

## Activation milestones

1. Register/login completed
2. First product saved
3. First affiliate link added or confirmed
4. First AI content generated
5. First content copied or exported
6. First HyperFrames script/composition created
7. First render queued and then completed/reviewed
8. Return visit within 7 days

## First 7 days launch plan

### Day 1-2: Setup + message alignment

- Publish Thai-first value proposition and sample workflows on landing page
- Verify dashboard CTA order: Add product → Generate AI content → Export/copy → HyperFrames
- Prepare manual outreach list of 20-30 creators

### Day 3-4: Guided activation sessions

- Run 10 assisted onboarding calls/chats
- Capture friction in onboarding checklist and feedback log
- Prioritize fixes that unblock milestone #2 to #5

### Day 5-7: Iterate and stabilize

- Ship copy improvements and safe onboarding clarifications
- Re-run guided onboarding with the next 15-20 users
- Review activation funnel drop-offs and update scripts/questions

## First 30 days growth loop

- Weekly cycle:
  1. Acquire 20-30 new targeted signups
  2. Run manual onboarding support for first-time users
  3. Collect Thai feedback and tag by activation milestone
  4. Ship top 3 friction fixes (copy, checklist, template clarity)
  5. Re-measure activation/retention next week
- Keep HyperFrames as optional conversion enhancer after content activation

## First 100 users acquisition channels (safe)

1. Thai creator communities (manual posting with transparent product value)
2. Existing affiliate/seller networks (warm outreach)
3. Manual referral invites from activated users (no automation)
4. Founder-led demos in Thai (short call + follow-up checklist)
5. Educational content about safe affiliate disclosure and workflow speed

## Content quality checklist

- Product data is verified by user before publish
- Caption/platform copy matches selected platform format
- No fabricated specs, testimonials, or fake reviews
- Tone is clear, concise, and Thai-first by default
- Affiliate disclosure is present and editable

## Affiliate compliance checklist

- Every affiliate post includes visible disclosure text
- No guaranteed income wording
- No exaggerated health/legal/financial claims
- User can edit generated content before posting

## Privacy boundaries

- Collect only product/content workflow data needed for activation metrics
- Do not collect private Shopee user data
- Do not log secrets, tokens, or database connection strings
- Keep analytics as documentation plan until explicitly approved
