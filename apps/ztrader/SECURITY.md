# Security Policy

## Supported Versions

Only the latest release of `ztrader` is supported for security updates.

| Version | Supported |
| ------- | --------- |
| v1.x.x  | Yes       |
| < v1.0  | No        |

## Reporting a Vulnerability

We take the security of `ztrader` seriously. If you believe you have found a security vulnerability in this repository or application, please do not report it publicly through Github issues. Instead, please follow these steps:

1. **Email us**: Send a detailed report to security@zeaz.dev.
2. **Details to include**:
   - Description of the vulnerability and its potential impact.
   - Detailed step-by-step instructions to reproduce the issue.
   - Any proof-of-concept code or payloads.
3. **Acknowledgment**: We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue.

Please keep details confidential until a patch has been published and released.
