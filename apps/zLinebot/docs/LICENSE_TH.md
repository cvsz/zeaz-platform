# สรุปสัญญาอนุญาต (TH)

โครงการนี้ใช้สัญญาอนุญาต MIT

- สามารถใช้งาน คัดลอก แก้ไข เผยแพร่ และจำหน่ายซอฟต์แวร์ได้
- ต้องคงประกาศลิขสิทธิ์และข้อความสัญญาอนุญาตไว้
- ซอฟต์แวร์ให้ใช้งานแบบ "ตามสภาพ" (ไม่มีการรับประกัน)
