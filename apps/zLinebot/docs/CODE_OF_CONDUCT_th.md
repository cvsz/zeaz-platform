# ข้อปฏิบัติชุมชน (TH)

## คำมั่นสัญญา
เรามุ่งสร้างสภาพแวดล้อมที่ปลอดภัยและเป็นมิตร ปราศจากการคุกคามและการเลือกปฏิบัติทุกรูปแบบ

## พฤติกรรมที่คาดหวัง
- สื่อสารด้วยความสุภาพและสร้างสรรค์
- โฟกัสที่ประเด็นงาน ไม่โจมตีตัวบุคคล
- ให้/รับฟีดแบ็กอย่างมืออาชีพ
- ยึดเป้าหมายโครงการและความปลอดภัยของผู้ใช้

## พฤติกรรมที่ไม่ยอมรับ
- การคุกคาม เหยียด หรือ hate speech
- การโจมตีส่วนบุคคล การปั่น การเปิดเผยข้อมูลส่วนตัวผู้อื่น
- พฤติกรรมรบกวนซ้ำๆ แม้ได้รับคำเตือนแล้ว

## การบังคับใช้
ผู้ดูแลโครงการสามารถลบคอนเทนต์ ปิด issue/PR หรือระงับบัญชีผู้ใช้งานได้

## การรายงาน
กรุณารายงานเหตุการณ์ผ่านช่องทางส่วนตัวของผู้ดูแลโครงการ (อีเมล/DM ที่ผู้ดูแลใช้งาน)
