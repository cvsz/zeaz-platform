# ZLineBot Presentation (TH)

## สไลด์ 1: ZLineBot คืออะไร
แพลตฟอร์มคอมเมิร์ซ AI สำหรับงานผ่านแชตและ API

## สไลด์ 2: ปัญหา
ทีมงานต้องการเครื่องมือรวมศูนย์สำหรับ chat commerce, billing และ compliance

## สไลด์ 3: โซลูชัน
สแตกเดียวสำหรับ products/cart/orders + LINE + admin + compliance

## สไลด์ 4: สถาปัตยกรรม
Multi-tenant Express API, realtime metrics แบบ event-driven และ deploy ได้ทั้ง Docker/Kubernetes

## สไลด์ 5: คุณค่าธุรกิจ
เริ่มระบบได้เร็วขึ้น เพิ่มโอกาส conversion และลดความเสี่ยงงานปฏิบัติการ

## สไลด์ 6: เดโม
สร้างสินค้า -> ถามบอท LINE -> สร้างออเดอร์ -> ดู metrics -> ทำ DSR

## สไลด์ 7: แผนพัฒนา
ทำระบบหลักให้เสถียร -> เพิ่มความน่าเชื่อถือด้าน payment/trust -> ขยายความสามารถ AI

## สไลด์ 8: การตัดสินใจถัดไป
เลือกรูปแบบ deployment และกำหนด baseline ด้าน security/SLA
