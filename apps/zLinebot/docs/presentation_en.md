# ZLineBot Presentation (EN)

## Slide 1: What is ZLineBot?
AI commerce platform for chat + API operations.

## Slide 2: Problem
Teams need unified tooling for chat commerce, billing, and compliance.

## Slide 3: Solution
Unified stack for products/cart/orders + LINE + admin + compliance.

## Slide 4: Architecture
Multi-tenant Express API, event-driven realtime metrics, Docker/Kubernetes deployment.

## Slide 5: Business Value
Faster launch, better conversion potential, lower operational risk.

## Slide 6: Demo Flow
Create products -> ask LINE bot -> place order -> observe metrics -> execute DSR.

## Slide 7: Roadmap
Stabilize core -> improve trust/payments -> scale intelligence.

## Slide 8: Next Decision
