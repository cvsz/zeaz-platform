# ZLineBot Roadmap (TH)

## ระยะที่ 1 (0-30 วัน)
ทำ API ให้เสถียร ปรับเอกสาร เพิ่มความแข็งแรงของ auth และปรับ UX แอดมิน

## ระยะที่ 2 (30-60 วัน)
ยกระดับความน่าเชื่อถือการชำระเงิน, webhook idempotency และ workflow compliance ตาม SLA

## ระยะที่ 3 (60-120 วัน)
นำ personalization/ranking ไปใช้จริง พร้อมระบบทดลองที่วัดผลได้

## ระยะที่ 4 (120-180 วัน)
ขยายสู่ระดับ enterprise: multi-region, SSO/SCIM และ partner API

## งานหนี้เทคนิค
- กระจายระบบ rate limiting
- รวม migration framework
- เพิ่ม CI test pyramid
- ลดช่องว่างฟีเจอร์ระหว่าง backend/admin
