# Security Policy

## Supported Versions

Currently, only the latest release of `zsticker` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerabilities within `zsticker`, please do **NOT** open a public issue.

Instead, please send a direct message or email to the repository owner with the details of the vulnerability. We will endeavor to respond within 48 hours to acknowledge receipt of the report and begin investigation.

We take all security vulnerabilities seriously and will work to release a patch and publish a security advisory as quickly as possible.
