# ZeaZ Platform — Go/No-Go Checklist

This checklist must be completed before initiating any deployment to a production environment.

## Go/No-Go Criteria
- [ ] Infrastructure/Code is in a stable state.
- [ ] All security scanners (Trivy, Semgrep) passed.
- [ ] Disaster Recovery runbooks are verified and updated.
- [ ] Required environment variables are validated.
- [ ] Operator signoff is obtained.
- [ ] Emergency rollback plan is documented and tested.

*Deployment is authorized only when all items are checked.*
