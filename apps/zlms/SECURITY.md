# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities to the parent repository's [SECURITY.md](../../SECURITY.md).
For critical issues, contact the security team directly through the ZeaZ Platform
GitHub security advisory process.

You can expect an initial response within 48 hours. We follow responsible disclosure:
we will work with you to understand the issue, develop a fix, and coordinate a release
before public disclosure.
